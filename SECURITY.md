# Security Policy

## Reporting a vulnerability

Please report vulnerabilities privately via GitHub Security Advisories:
open a private report at
https://github.com/Vidia-Tools/Vidia-Open-Studio/security/advisories/new

Do not open public issues for security bugs.

Response is best-effort. This is a maintainer-run open source project, so
there is no guaranteed response time, but reports are taken seriously and
addressed as soon as possible.
